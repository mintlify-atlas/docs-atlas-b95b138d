ScholarXIV's AI Chat feature provides an intelligent research assistant that helps you understand papers, synthesize findings, and explore research topics in depth.

## Overview

The AI assistant uses Google's Gemini 2.0 Flash model to provide comprehensive, evidence-based responses grounded in academic research. It can search papers, analyze content, and help you navigate complex research topics.

<Note>
  The AI assistant requires you to provide your own Gemini API key. Set it up in Settings to start using AI features.
</Note>

## Getting Started

<Steps>
  <Step title="Set up your API key">
    Click the settings icon in the main input box and enter your Gemini API key. The key is securely stored and used only for your requests.
  </Step>
  
  <Step title="Start a conversation">
    Type your research question or topic in the chat input. The AI will help you explore it using arXiv papers.
  </Step>
  
  <Step title="Select relevant papers">
    You can select papers from search results to give the AI specific context about what you're researching.
  </Step>
  
  <Step title="Get comprehensive answers">
    The AI will search for relevant papers and synthesize findings into clear, actionable insights.
  </Step>
</Steps>

## How It Works

The AI assistant follows a structured research workflow:

### System Instructions

From `ai_chat/+server.ts:7-8`, the AI is configured with this core purpose:

> "You are a research assistant helping people navigate and understand research papers more. You are inside an arxiv repository and the users will often send you a list of papers they have selected along with your previous conversation history."

### Research Process

<Steps>
  <Step title="Context gathering">
    The AI receives:
    - Your current prompt/question
    - Previously selected papers
    - Conversation history
  </Step>
  
  <Step title="Analysis">
    The AI analyzes your question and determines the best approach to answer it using available research.
  </Step>
  
  <Step title="Response generation">
    Generates a comprehensive response formatted in markdown with proper citations and structure.
  </Step>
</Steps>

## Working with Selected Papers

The AI can work with papers you've specifically selected:

<Steps>
  <Step title="Select papers">
    Click the checkbox on any paper in your search results to select it for AI analysis.
  </Step>
  
  <Step title="Ask questions">
    The AI will automatically consider these papers when responding to your questions.
    
    Examples:
    - "Summarize these papers"
    - "What are the main findings?"
    - "How do these papers relate to each other?"
  </Step>
  
  <Step title="Context updates">
    The AI checks your selected papers with each new message, so you can change your selection as you explore.
  </Step>
</Steps>

## Best Practices

### For Better Results

1. **Be specific**: Instead of "Tell me about AI", try "What are the latest advances in transformer architectures for natural language processing?"

2. **Use context**: Select relevant papers before asking questions to get more focused, accurate responses

3. **Follow up**: Ask clarifying questions and dig deeper into topics that interest you

4. **Combine approaches**: Mix general questions with paper-specific queries for comprehensive understanding

### Response Formatting

The AI formats responses using markdown for clarity:

- **Headers** for major sections
- **Bullet points** for key findings
- **Tables** when comparing data
- **Code blocks** when discussing implementations
- **Citations** linking to specific papers

<Warning>
  If you haven't set up your API key, you'll see an error message. Click the settings icon to add your Gemini API key.
</Warning>

## Privacy & Security

- Your API key is stored securely in the database
- Conversations are private to your account
- The AI only accesses publicly available arXiv papers
- No training data is collected from your chats

## Technical Details

Implementation reference from `ai_chat/+server.ts:32-62`:

- Uses OpenAI SDK with Gemini base URL
- Model: `gemini-2.0-flash`
- Context window includes selected papers and conversation history
- Responses are streamed for better user experience
- Automatic error handling for invalid API keys
