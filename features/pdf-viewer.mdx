## PDF Rendering

The viewer uses PDF.js for high-quality rendering:

<Steps>
  <Step title="Document loading">
    PDF is fetched from arXiv servers using the paper's PDF link.
  </Step>
  
  <Step title="Page rendering">
    Each page is rendered to a canvas element with proper scaling.
    
    ```typescript
    // From pdfviewer.svelte:24-68
    const pdf = await pdfjsLib.getDocument(pdfPreviewState.pdfLink).promise;
    
    for (let i = 1; i <= pdf.numPages; i++) {
      const page = await pdf.getPage(i);
      const viewport = page.getViewport({ scale });
      
      // Fit to container width
      const scale = container.clientWidth / viewport.width;
      const scaledViewport = page.getViewport({ scale });
      
      // Render to canvas
      await page.render({
        canvasContext: context,
        viewport: scaledViewport
      }).promise;
    }
    ```
  </Step>
  
  <Step title="Display">
    All pages are stacked vertically for continuous scrolling.
  </Step>
</Steps>

## Technical Implementation

### PDF.js Integration

The viewer uses PDF.js legacy build for maximum compatibility:

```typescript
// From pdfviewer.svelte:26-29
const pdfjsLib = await import('pdfjs-dist/legacy/build/pdf');

pdfjsLib.GlobalWorkerOptions.workerSrc =
  'https://cdnjs.cloudflare.com/ajax/libs/pdf.js/5.4.149/pdf.worker.min.mjs';
```

### State Management

PDF viewer state is managed globally:

- `pdfPreviewState.pdfLink`: Current PDF URL
- `pdfPreviewState.paper`: Paper metadata
- `pdfPreviewState.currentScale`: Current zoom level
- `pdfPreviewState.closePDF()`: Cleanup function
- `pdfPreviewState.changeScale()`: Zoom control

### Performance Optimizations

## Troubleshooting

<Tabs>
  <Tab title="PDF Won't Load">
    **Issue**: PDF viewer shows loading indefinitely
    
    **Solutions**:
    - Check your internet connection
    - Verify the paper exists on arXiv
    - Try refreshing the page
    - Check browser console for errors
  </Tab>
  
  <Tab title="Blurry Text">
    **Issue**: Text appears blurry or pixelated
    
    **Solutions**:
    - Try a different zoom level (1.0x or 1.5x usually best)
    - Ensure your browser zoom is at 100%
    - Check if your display scaling is set correctly
  </Tab>
  
  <Tab title="Slow Scrolling">
    **Issue**: Scrolling is laggy on long papers
    
    **Solutions**:
    - Use a lower zoom level (0.9x or 0.5x)
    - Close other browser tabs to free memory
    - Try a different browser (Chrome/Edge recommended)
  </Tab>
</Tabs>

<Warning>
  Very large papers (100+ pages) may take longer to load. The viewer renders all pages at once for smooth scrolling.
</Warning>
